Detailed Description

An abstract visitor that the generated code calls into that we use to implement various features. Fields that are not members of oneofs are always visited. Members of a oneof are only visited when they are the set oneof case value on the "other" proto. The visitOneofNotSet method is invoked if other's oneof case is not set.
