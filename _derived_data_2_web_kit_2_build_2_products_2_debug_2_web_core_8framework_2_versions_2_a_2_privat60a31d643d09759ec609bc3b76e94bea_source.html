104  * This is the format of the information CoreText gives us about each run:
105  *
106  * ----->X (Paint glyph position) X (Paint glyph position) X (Paint glyph position)
107  * / 7 7 7
108  * / / / /
109  * (Initial advance) / / (Glyph origin) / (Glyph origin) / (Glyph origin)

115  *

117  *
118  *

120  *
121  * ----->X------------------------->X------------------------->X
