44  // The canSuspendForDocumentSuspension() function is used by the caller if there is a choice between suspending
45  // and stopping. For example, a page won't be suspended and placed in the back/forward

48  // However, the suspend function will sometimes be called even if canSuspendForDocumentSuspension() returns false.
