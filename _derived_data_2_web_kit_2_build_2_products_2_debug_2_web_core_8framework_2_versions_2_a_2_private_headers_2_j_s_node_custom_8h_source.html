50 // In the C++ DOM, a node tree survives as long as there is a reference to its
51 // root. In the JavaScript DOM, a node tree survives as long as there is a
52 // reference to any node in the tree. To model the JavaScript DOM on top of
53 // the C++ DOM, we ensure that the root of every tree has a JavaScript wrapper.
