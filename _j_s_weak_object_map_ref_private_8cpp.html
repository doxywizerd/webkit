JSWeakObjectMapRef JSWeakObjectMapCreate ( JSContextRef  ctx,
void *  data,
JSWeakMapDestroyedCallback  destructor 
)

Creates a weak value map that can be used to reference user defined objects without preventing them from being collected.

Parameters
ctxThe execution context to use.
dataA void* to set as the map's private data. Pass NULL to specify no private data.
destructorA function to call when the weak map is destroyed.
Returns
A JSWeakObjectMapRef bound to the given context, data and destructor. The JSWeakObjectMapRef can be used as a storage mechanism to hold custom JS objects without forcing those objects to remain live as JSValueProtect would.
