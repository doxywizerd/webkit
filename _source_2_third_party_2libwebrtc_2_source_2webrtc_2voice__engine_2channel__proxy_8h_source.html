123  // Thread checkers document and lock usage of some methods on voe::Channel to
124  // specific threads we know about. The goal is to eventually split up
125  // voe::Channel into parts with single-threaded semantics, and thereby reduce
126  // the need for locks.
127  rtc::ThreadChecker worker_thread_checker_;
