enum WebKitHardwareAccelerationPolicy

WebKitHardwareAccelerationPolicy: : Hardware acceleration is enabled/disabled as request by web contents. : Hardware acceleration is always enabled, even for websites not requesting it. : Hardware acceleration is always disabled, even for websites requesting it.

Enum values used for determining the hardware acceleration policy.
