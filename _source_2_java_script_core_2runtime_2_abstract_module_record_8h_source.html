142  // Currently, we don't keep the occurrence order of the import / export entries.
143  // So, we does not guarantee the order of the errors.
144  // e.g. The import declaration that occurr later than the another import declaration may
145  // throw the error even if the former import declaration also has the invalid content.
146  //
147  // import ... // (1) this has some invalid content.
148  // import ... // (2) this also has some invalid content.
149  //
150  // In the above case, (2) may throw the error earlier than (1)
151  //
152  // But, in all the cases, we will throw the syntax error. So except for the content of the syntax error,
153  // there are no difference.
154
