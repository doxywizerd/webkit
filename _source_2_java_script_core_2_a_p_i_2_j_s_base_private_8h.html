JS_EXPORT void JSReportExtraMemoryCost ( JSContextRef  ctx,
size_t  size 
)

Reports an object's non-GC memory payload to the garbage collector.

Parameters
ctxThe execution context to use.
sizeThe payload's size, in bytes. Use this function to notify the garbage collector that a GC object owns a large non-GC memory region. Calling this function will encourage the garbage collector to collect soon, hoping to reclaim that large non-GC memory region.
