34 // What is MacroAssembler::print()?
35 // ===============================
36 // The MacroAsssembler::print() makes it easy to add print logging
37 // from JIT compiled code, and can be used to print all types of values
38 // at runtime e.g. CPU register values being operated on by the compiled
39 // code.
40 //
41 // print() is built on top of MacroAsssembler::probe(), and hence
42 // inserting logging in JIT compiled code will not perturb register values.
43 // The only register value that is perturbed is the PC (program counter)
44 // since there is now more compiled code to do the printing.
45 //
46 // How to use the MacroAssembler print()?
47 // =====================================
48 // 1. #include "MacroAssemblerPrinter.h" in the JIT file where you want to use print().
49 //
50 // 2. Add print() calls like these in your JIT code:
51 //
52 // jit.print("Hello world\n"); // Emits code to print the string.
53 //
54 // CodeBlock* cb = ...;
55 // jit.print(cb, "\n"); // Emits code to print the pointer value.
56 //
57 // RegisterID regID = ...;
58 // jit.print(regID, "\n"); // Emits code to print the register value (not the id).
59 //
60 // // Emits code to print all registers. Unlike other items, this prints
61 // // multiple lines as follows:
62 // // cpu {
63 // // eax: 0x123456789
64 // // ebx: 0x000000abc
65 // // ...
66 // // }
67 // jit.print(AllRegisters());
68 //
69 // jit.print(MemWord<uint8_t>(regID), "\n"); // Emits code to print a byte pointed to by the register.
70 // jit.print(MemWord<uint32_t>(regID), "\n"); // Emits code to print a 32-bit word pointed to by the register.
71 //
72 // jit.print(MemWord<uint8_t>(Address(regID, 23), "\n"); // Emits code to print a byte at the address.
73 // jit.print(MemWord<intptr_t>(AbsoluteAddress(&cb), "\n"); // Emits code to print an intptr_t sized word at the address.
74 //
75 // jit.print(Memory(reg, 100), "\n"); // Emits code to print a 100 bytes at the address pointed by the register.
76 // jit.print(Memory(Address(reg, 4), 100), "\n"); // Emits code to print a 100 bytes at the address.
77 //
78 // // Print multiple things at once. This incurs the probe overhead only once
79 // // to print all the items.
80 // jit.print("cb:", cb, " regID:", regID, " cpu:\n", AllRegisters());
81 //
82 // The type of values that can be printed is encapsulated in the PrintArg struct below.
83 //
84 // Note: print() does not automatically insert a '\n' at the end of the line.
85 // If you want a '\n', you'll have to add it explicitly (as in the examples above).
