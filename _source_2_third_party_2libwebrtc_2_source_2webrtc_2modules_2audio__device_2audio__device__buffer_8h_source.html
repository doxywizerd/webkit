125  // Called periodically on the internal thread created by the TaskQueue.
126  // Updates some stats but dooes it on the task queue to ensure that access of
127  // members is serialized hence avoiding usage of locks.
128  // state = LOG_START => members are initialized and the timer starts.
129  // state = LOG_STOP => no logs are printed and the timer stops.
130  // state = LOG_ACTIVE => logs are printed and the timer is kept alive.
