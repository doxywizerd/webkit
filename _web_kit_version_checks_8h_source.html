31 /*
32  Version numbers are based on the 'current library version' specified in the WebKit build rules.
33  All of these methods return or take version numbers with each part shifted to the left 2 bytes.
34  For example the version 1.2.3 is returned as 0x00010203 and version 200.3.5 is returned as 0x00C80305
35  A version of -1 is returned if the main executable did not link against WebKit.
36 
37  Please use the current WebKit version number, available in WebKit/mac/Configurations/Version.xcconfig,
38  when adding a new version constant.
39 */
