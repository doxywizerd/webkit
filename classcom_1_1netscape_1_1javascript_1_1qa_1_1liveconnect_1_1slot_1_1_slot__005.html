Detailed Description

Create a JS object/ Set indexed members of the object using JSObject.setSlot. Get indexed members of the object again using getSlot.

This covers the following JavaScript -> Java cases through getSlot:

Java type JavaScript type
JSObject the JavaScript object it refers to
See also

netscape.javascript.JSObject::setSlot
