◆ webkit_web_view_get_settings()

WEBKIT_API WebKitSettings* webkit_web_view_get_settings ( WebKitWebView *  webView)

webkit_web_view_get_settings: : a WebKitWebView

Gets the WebKitSettings currently applied to . If no other WebKitSettings have been explicitly applied to with webkit_web_view_set_settings(), the default WebKitSettings will be returned. This method always returns a valid WebKitSettings object.
